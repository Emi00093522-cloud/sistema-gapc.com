# Sistema GAPC - Comunidad Ahorra

Sistema de Gestión para Grupos de Ahorro y Préstamo Comunitario

## Instalación
1. pip install -r requirements.txt
2. Configurar .env con datos de BD
3. streamlit run app/main.py
